## 프로젝트 개요
- 멋쟁이 사자처럼 백엔드스쿨 3기 - 스프링부트 수업 및 기능 구현 미션을 위한 프로젝트로 인스타ID 기반 익명호감표시 서비스 구현.
- 본인의 인스타ID를 입력하면, 현재 자신에게 이성적으로 호감을 느끼고 있는 사람이 몇명인지 알 수 있다.
- 나를 좋아하는 상대방이 나의 어떤 매력포인트 때문에 좋아하는지 알 수 있으며, 상대방의 성별도 알 수 있다.
